Hello and Welcome to React My Todos application
Installation Instaraction:

- 1st Install NodeJS: got to the https://nodejs.org/en/download/
  and download specific to your browsers
- 2nd: create a git repository and clone to your local disktop using either github desktop or visual studio commands
- 3rd : install vite using your command line by running under the git repository directory: npm create vite@latest ./ -- --template react
  then install other dependances by running: npm install
- 4th: Run the development server: npm run dev
